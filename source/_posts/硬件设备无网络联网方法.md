# 硬件设备无网络联网方法
先将硬件设备与笔记本直连

将虚拟机网络禁用

1.打开WIFI的共享网络
![](img\1741683288350-2d114da6-aeb4-42f0-ba20-c3514e5feb85.png)

2.配置网卡的网络
![](img\1741683335601-0aa71e70-312f-40be-a184-a0866819df03.png)3.将硬件设备的ip改到对应的网段 网关改为以太网IP
![](img\1741683433858-0b267c35-1085-4501-99d4-1515afe14305.png)

联网成功

# 可能遇到的问题
还是无法上网

输入ip route查看路由
![](img\1742887921066-e68f0b13-e5b3-4961-8e11-665571f32c29.png)

发现有多个路由

删除不需要的

sudo ip route del default via 192.168.1.1 dev enp129s0

再次ping 连接成功
![](img\1742887953208-156f03bb-bb98-4ab6-9fc3-d633ef30b2c7.png)
